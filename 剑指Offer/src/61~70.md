# 61. 扑克牌中的顺子

[61. 扑克牌中的顺子](https://leetcode-cn.com/problems/bu-ke-pai-zhong-de-shun-zi-lcof/)

```
class Solution {
public:
    bool isStraight(vector<int>& nums) {
		sort(nums.begin(),nums.end());
		int i = 0;
		while(i < nums.size() && nums[i] == 0)
			i++; //@ i指向非零元素
		
		//@ 全为0时,特判
		if(i == nums.size())
			return true;
		
		//@ 出现相同的牌，返回false
		for(int j = i;j < 4;j++)
			if(nums[j] == nums[j+1])
				return false;
			
		//@ 最大的牌和非0最小牌不能超过4
		return nums[4] - nums[i] <= 4;
    }
};
```
