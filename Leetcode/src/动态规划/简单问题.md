# [70. 爬楼梯](https://leetcode-cn.com/problems/climbing-stairs/)

```
class Solution {
public:
    int climbStairs(int n) {
        if(n <= 2)
            return n;
        int pre2 = 1,pre1 = 2,res;
        for(int i = 2;i < n;i++)
        {
            res = pre1 + pre2;
            pre2 = pre1;
            pre1 = res; 
        }
        return res;
    }
};
```
